Documentation for the Reversible Computing 2016 (RC16) conference
work-in-progress report "Elements of a Reversible Object-Oriented
Language" by Ulrik Pagh Schultz Holger Bock Axelsen.

This directory contains the source files of the xtext-based Joule
implementation used in the paper as well as the example shown in the
paper, allowing all examples to be compiled to Janus code and run
e.g. using the publicly available DIKU TOPPS implementation of Janus.
